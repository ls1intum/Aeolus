++++++++++++++++++
How do I run this?
++++++++++++++++++

It's easy! Just run the following command:

.. code-block:: bash

    cd cli/
    python3 main.py

or if you prefer docker:

.. code-block:: bash

    docker run --rm -v "./:/tmp" ghcr.io/ls1intum/aeolus/cli:nightly --help
